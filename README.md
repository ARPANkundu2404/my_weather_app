1. **Clone the repository**
   ```
   git clone https://github.com/ARPANkundu2404/my_weather_app.git
   cd my_weather_app
   ```

2. **Install dependencies**
   ```
   npm install
   ```

3. **Configure API Key**
   - Create a `.env` file in the root directory:
     ```
     VITE_API_KEY=your_openweather_api_key

   ```
